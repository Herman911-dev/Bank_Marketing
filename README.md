# Bank_Marketing
Dans un contexte bancaire, la mise en place de campagnes marketing efficaces  est essentielle pour maximiser la rentabilité et améliorer l'expérience client. Ce projet vise à utiliser des  techniques d’exploiratoire de donnees et des analyses statistiques afin d'aider la banque à décider quels clients  cibler pour souscrire à des dépôts à terme.
